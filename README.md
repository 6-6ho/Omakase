[수악중독](https://www.youtube.com/minipole) 고등학교, 중학교 수학 관련 개념 설명은 이 채널이 최강이다! 강의를 아주 스무스하게 잘 하신다.

[Educast 해석학개론](https://educast.com/17.330/) 설수리 (당시) 박사과정생이 찍은 해석학 영상. 개론 수준이다.
